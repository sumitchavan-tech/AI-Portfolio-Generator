# AI Portfolio Generator

## Overview

AI Portfolio Generator is a Flask-based web application that analyzes resumes using AI and automatically generates professional portfolio websites.

## Features

* Resume PDF Upload
* AI Resume Analysis
* Resume Score Generation
* Career Recommendations
* Skill Gap Analysis
* Learning Recommendations
* Portfolio Website Generation
* PDF Report Generation
* QR Code Generation
* ATS Dashboard
* Render Deployment

## Tech Stack

* Python
* Flask
* Groq API
* LangChain
* Jinja2
* HTML/CSS
* ReportLab
* Git
* GitHub
* Render

## Deployment

Live application hosted on Render.
